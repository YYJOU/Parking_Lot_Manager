# 小專題題目 : 停車場管理系統

# 專案簡介：
### 管理多層樓停車場的車位、車輛進出、月租車、收費與資料保存

# 主要功能：
### 車輛進出場
### 月租車管理
### 停車場狀態查詢
### 管理員模式（含密碼）
### 收費與盈收統計
### 資料匯入/匯出


# 程式如何執行：
### * 在github的Download分支點開Code
### * 點擊Download ZIP進行下載
### * 將檔案解壓縮
### * 於Visual Sudio選擇開啟本機資料夾
### * 在畫面中找到檔案
### * 開啟後即完成安裝
### * 打開後點選執行
![run](https://github.com/user-attachments/assets/04abe41d-0a1a-4161-affc-be9763cf4352)
### * 即可開始執行
